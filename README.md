# 🌌 Solar System Simulation

A visually engaging simulation of the solar system using **HTML** and **CSS**.

## 🛠️ Skills & Tools Used

- **Languages:** HTML5, CSS3  
- **Tools:** VS Code (or any code editor), Web Browser  
- **Skills:**  
  - CSS Animation  
  - Flexbox & Positioning  
  - UI Design  
  - Basic Understanding of Planetary Motion
